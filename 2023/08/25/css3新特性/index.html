# 位移

transorm: translate 属性
x 和 y 轴一起位移 50px，使用 % 时注意是参考自身的位置位移

注意：

  1. 位移与相对定位很相似，都不脱离文档流，不会影响到其他元素
  2. 与相对定位的区别：相对定位的百分比值参考的是其父元素；位移的百分比值参考的是其自身
  3. 浏览器针对位移有优化，处理位移的效率更高
  4. 可以链式编写，参考上述代码
  5. 位移对行内元素无效
  6. 位移配合定位可实现元素水平居中，如：

.box {
position: absolute;
left:50%;
top:50%;
transform:translate(-50%,-50%)
}

# 空间与景深

  1. 首先要找到 3D 变换元素的父元素并开启 3D 空间，即： transform-style: preserve-3d ；其中 transform-style 的默认属性为 flat ，也就是 2D。
  2. 设置景深，即指定观察者与 **z=0** 平面的距离，通过 perspective 属性，值为长度值（默认值为 none），单位为 px 等。可以获得透视效果，看起来更加立体。也是给要 3D 变换的元素的父元素设置。

# 透视点（观察者）

perspective-origin 属性，值为长度值。要在给 3D 变换元素的父元素设置。通常情况下并不需要去更改透视点的位置。

# 过渡

使用 transition 属性。
给需要进行过度变化的元素通过 transition-property 设置过渡属性，在通过 transition-duration 设置过渡的时间。

  1. transition-property 可以设置多个属性；
  2. transition-duration 可以设置多个值；
  3. 过渡只能影响可以设置数值的属性。
  1. transition-delay 属性可以设置过渡的延迟，值为 s。
  2. transition-timing-function 属性可以设置过渡的方法，默认值为： ease （平滑过渡）。可选值为：
    1. linear （匀速过渡）
    2. ease-in （先慢后快）
    3. ease-out （先快后慢）
    4. ease-in-out （类似 ease）
    5. steps(num) 分步过渡，具体分为 num 的步数
    6. cubic-bezier() 贝塞尔曲线

# 基本使用

通过 animation-name 设置动画方法，应用到元素当中。还可以设置其他属性，如： animation-duration 、 animation-delay 等。

通过 @keyframes function 的方法设置动画方法，但只关注第一帧或最后一帧。

# 其他属性

属性说明
animation-timing-function动画播放的方式，默认值为 linear 。可选值为 steps() 、 ease 、 cubic-bezier() 等
animation-iteration-count动画播放的次数，默认播放一次，可选值为： infinite 等
animation-direction动画的播放方向，可选值为： reverse 、 alternate 等
animation-fill-mode动画以外的状态，可选值为： forwards 停在最后一帧、 backwards
animation-play-state动画的播放状态，可选值为： paused 、 running

# 过渡与动画的区别

  1. 动画不需要触发条件，而过渡需要触发条件；
  2. 动画能进行精细的调整，而过渡不能。

# 伸缩盒模型（弹性盒子）

通过 display: flex 属性将元素设置为伸缩容器，即开启 flex 布局。可选值有： flex 、 inline-flex 。
设置 dispaly:inline-flex 可以将元素变为行内块。

  1. 仅伸缩容器内的子元素成为伸缩项目，而子元素的子元素则不会；
  2. 伸缩项目全部都是块级元素。
