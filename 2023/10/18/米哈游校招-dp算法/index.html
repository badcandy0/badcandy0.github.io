# 题目描述

米小游都快保底了还没抽到希儿，好生气哦！只能打会活动再拿点水晶了。米小游和世界第一可爱的魔法少女 TeRiRi 正在打 BOSS，当 BOSS 血量小于等于 0 时，BOSS 死亡。TeRiRi 的血量为 h，有一套牌，在一轮中，他会按顺序一张一张的将卡牌打出，套牌中有两种卡牌：

  1. 时来运转，获得 x 个幸运币
  2. 幸运一掷造成 x 点伤害，并投掷所有幸运币，造成等于所有幸运币掷出的点数之和的伤害

幸运币可以等概率的投掷出 1～6 之间的点数。米小游想知道，TeRiRi 的套牌在一轮内击杀 BOSS 的概率

# 输入格式

第一行输入两个整数 n (1<n<100)，h (1<h<109)，分别表示卡牌张数和 BOSS 血量，接下来 n 行每行首先输入两个整数 t (1<=<=<=t$<=$2)

x(1<=<=<=x$<=$10)，t 为 1 表示卡牌为时来运转，t 为 2 表示卡牌为幸运一掷。

# 思路

  1. 首先判断是否为幸运一掷，如果是的话，BOSS 血量直接减去造成的 x 伤害，然后计算幸运币所造成的伤害总期望；如果为否的话，则不减去造成的 x 伤害
  2. 计算所有幸运币造成的伤害期望 (重点)

重点就在于如何计算幸运币造成的伤害期望。

# 代码

#include<iostream>
#include<cmath>
#include<cstring>
using namespace std;

int main() {
    int n,h,luck=0,tmp=0;
    cin >> n >> h;
    double p = 0;
    for (int i = 0; i < n; i++)
    {
        int t,x;
        cin >> t >> x;
        if(t == 1) tmp += x; //缓存幸运金币 如果最后几张为时来运转则不计入总伤害
        else {
            h -= x; // BOSS血量减去幸运一掷的固定伤害
            luck += tmp; // 将缓存的幸运金币取出
            tmp = 0; // 重制缓存
        }
    }
    if (luck>=h) p = 1; //如果幸运币总数大于等于boss剩余血量，则必击杀
    else if (luck*6 < h) p=0; //如果幸运币总数乘6小于boss剩余血量，则不可能击杀
    else {
        // 计算所有幸运币造成伤害的总期望
        double arr[luck][luck*6];
        memset(arr, 0, sizeof(arr));
        for(int i=0;i<6;i++) arr[0][i]=1.0/6;

        // for(int i=1;i<luck;i++) {
        //     for(int j=i;j<6;j++) 
        //         for(int k=i;k<=j;k++) arr[i][j]+=arr[i-1][k]/6; //防止数组索引为负 arr[0][0]~arr[5][5]单独处理
        //     for(int j=6;j<6*luck;j++) 
        //         for(int k=1;k<7;k++) arr[i][j]+=arr[i-1][j-k]/6;
        // }
        for (int i=0;i<luck;i++) {
            for (int j=i;j<6*i;j++) { // 第i次可能情况只有i～6*i
                for (int k=1;k<=6;k++) {
                    if (j-k>0)  // 只有大于0才能状态转移
                      arr[i][j] = arr[i-1][j-k] / 6;
                }
            }
        }

        for(int i=luck*6-1;i>=h-1;i--) {
            p+=arr[luck-1][i];
        }
    }
    cout<<p<<endl;
    return 0;
}
  • 时间复杂度：O (n3)
